En este proyecto se ha desarrollado un programa que permite gestionar una lista de vuelos y filtrarlos según un rango de fechas dado. El objetivo es mostrar primero todos los vuelos disponibles y, después, únicamente aquellos que cumplen con las fechas indicadas. Para conseguirlo, se han creado varias clases que se encargan de organizar y distribuir el trabajo de forma sencilla y clara.

En primer lugar, se creó la clase Vuelo, que representa un vuelo individual. Esta clase funciona como una plantilla que contiene toda la información relevante de cada vuelo: su identificador, el nombre del vuelo, la compañía, el lugar de salida, el lugar de llegada y las fechas de salida y llegada. También incluye métodos para obtener cada uno de estos datos y un método toString() que permite mostrar la información del vuelo de manera ordenada cuando se imprime en pantalla. En resumen, esta clase sirve para almacenar y mostrar los datos de un vuelo de forma estructurada.

En segundo lugar, se diseñó la clase FiltroVuelos, que es la encargada de realizar toda la lógica relacionada con la validación y el filtrado de los vuelos. Esta clase contiene varios métodos que verifican que los datos sean correctos. Por ejemplo, se comprueba que la lista de vuelos no sea nula, que las fechas del rango sean válidas y que ningún vuelo tenga una fecha de llegada anterior a la de salida. Además, esta clase incluye el método principal de filtrado, que analiza cada vuelo y decide si su fecha de salida se encuentra dentro del rango solicitado. Si es así, lo incluye en la lista filtrada. Finalmente, los vuelos filtrados se devuelven ordenados por fecha de salida para una mejor visualización.

Por último, se encuentra la clase Main, que es la encargada de ejecutar todo el programa. En ella se crea una lista de vuelos de ejemplo mediante un método dedicado a ello. Después, esta lista completa se muestra al usuario para que pueda ver todos los vuelos disponibles. A continuación, se aplica el filtro utilizando la clase FiltroVuelos, y se muestran en pantalla únicamente los vuelos que cumplen con el rango de fechas definido. De esta manera, el programa permite comparar los vuelos totales con los filtrados de forma clara y directa.
